Établir un budget avec MMEX

Options d'établissement de budget

L'entrée de menu Affichage → Budgets en années fiscales peut aussi être utilisée pour passer d'une vue à l'autre.

MMEX permet d'établir un budget annualisé et/ou mensualisé. Ceci peut vous permettre de comparer votre budget à votre consommation réelle.

Les budgets peuvent être affichés en années civiles ou en années fiscales. Utilisez la boîte de dialogue pour paramétrer le début d'une année financière, ainsi que les paramètres permanents.

Créer un nouveau budget

Pour établir un budget, faites un clic droit sur le nœud 'Planificateur budgétaire' et ajoutez un budget mensualisé ou annualisé.

Éditer les catégories de budget

Dès qu'une année a été ajoutée, sélectionnez-la pour afficher le planificateur budgétaire.

Chaque catégorie peut être éditée en double-cliquant sur celle-ci. Cela doit être fait pour toutes les catégories.

Panneau de configuration du budget

Cela devient le budget pour cette année là. Les mois et/ou les années qui suivent peuvent alors en découler.

Le total est affiché pour chaque catégorie principale.

Lors de la configuration d'un budget, les totaux peuvent êtres déactivés en allant dans le menu: Affichage → Configuration des Budgets: Sans totaux.

Rapports des budgets

Il est possible de comparer l'argent dépensé par rapport au budget prévisionnel en utilisant les rapports sous 'Budget'.
